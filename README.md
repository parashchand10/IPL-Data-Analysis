### Project Overview
This project provides a structured and clean analysis of Indian Premier League (IPL) match data. It serves as a comprehensive diagnostic tool for understanding match dynamics, player performances, and team statistics through exploratory data analysis (EDA) and visualization.

Key Features
1. Data Cleaning: Comprehensive handling of missing values by imputing numeric data with the mean and categorical data with the mode.

2. Feature Engineering: Derived new metrics such as total_runs and total_wickets for every match to provide deeper insights into high-scoring and high-intensity games.

3. Exploratory Data Analysis (EDA): Statistical summaries and correlation heatmaps to identify relationships between match variables.

4. Visualization: Implementation of targeted charts using Seaborn and Matplotlib to diagnose performance trends.

### Technologies Used
Language: Python

Libraries:

1. Pandas: Data manipulation and analysis.

2. Matplotlib & Seaborn: Advanced data visualization and heatmaps.
